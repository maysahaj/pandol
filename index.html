GENERAL INFO:
Why Using Coordinator app?

You can collect coordinate data from land on whatever coordinate system you want, choosing from defined systems.
You can collect GPS coordinates.
You can view a coordinate data on the map and also you can search by address and see that on the map.(You can get directions to that point by clicking the red marker shown on the map).
At the same time, you can learn the altitude value of the point and the distance between the point and your current location by long pressing on the map.
Also you can see the distance and the elevation difference between two points created on the map.
You can convert an existing coordinate into another coordinate system.
You can do this by typing coordinates manually or opening an XLS file which contains coordinates.
You can display circumference and area measurements of some coordinates by opening the file which created before using this app.
You can measure circumference and area for some coordinates which determined manually on the map. Using this app, you can display a KML file on the map.
Coordinator app contains several useful tools for people interested in coordinates, maps and geographic informations systems (GIS).

usage:

Under the menu of"COLLECT COORDINATE" , there is the list of some countries whose coordinate systems are defined. After selecting the desired country, the coordinate systems for that country is shown. After selecting the desired coordinate system from the defined systems, the collection operation of the coordinate data from site can be tarted.

On the data collection screen, there is Google Map on the top and the current point of you is located on the map. There four different view options for Google Maps. The desired one can be selected. Usind "Add" button, the coordinate data of your current point is added to the temporary list which is at the bottom of the screen.

The points which are desired to delete can be deleted by cliking the "Delete" button after checking the box near the coordinate data. By pressing the "Save" button, you can save the temporary list to your mobile phone/tablet.
At first save, a folder named "My Coordinates" is created. All files are saved as txt format in that folder. The coordinate data in the temporary list can be send as an e-mail by clicking the "E-Mail" button.

Under the menu of "FIND COORDINATES", there is the list of some countries whose coordinate systems are defined. After selecting the desired country, the coordinate systems for that country is shown. After selecting the desired coordinate system from the defined systems, a coordinate data can be seen on the Google Map. After entering the coordinate data to the related fields and cliked to the "Go" button, a marker on the map is seen and the distance of that point to your current location in kilometers is shown under the map. In this menu, searching by adress can be made and also the distance of that point to your current location is shown.

Under the menu of "OPEN FILE", there is the list of some countries whose coordinate systems are defined. After selecting the desired country, the coordinate systems for that country is shown. After selecting the desired coordinate system from the defined systems, the points collected before using this app can be displayed. Also, the circumference and area values for those points are displayed here.

Using "OPEN KML" tool, a KML file, saved to the mobile device before, can be displayed over the map. But if the file size is too large, the file can not be opened.

By "CALCULATE AREA" tool, some points on the map are marked by long pressing on the map. After marking at least three points, when pressing "Evaluate" button, circumference and area measurements are displayed on the screen.

By"Your Distance To A Point"tool under the menu of "MEASURE DISTANCE", you long press on the map and you can get the longitude and latitude values of that point(WGS84), the elevation value of that point and the air distance of that point to your current location. By clicking the "Clear" button, you can clear the existing values written on the screen.

By"Distance Between Two Points" tool under the menu of"MEASURE DISTANCE", you long press on the map for two points, you can get the elevation difference between those two points and the air distance between two points.

By"Length of a Path" tool under the menu of "MEASURE DISTANCE", you long press on the map for several points and you draw a path. After adding points and clicking the "Evaluate" button you can see the length of this path

Under the menu of "CONVERT COORDINATES", there is the list of some countries whose coordinate systems are defined. There two options for conversion. First, by writing only one coordinate. Here in"Existing Values", the coordinate system and central meridian are selected and the coordinate data is entered. In "To Be Converted" side, the coordinate system and central meridian for converting are selected. And you see the result by cliking the"Evaluate" button. The second method is to convert from XLS excel file. You can use the first row for title. The application starts reading from second row. The Y values must be in the first column and the X values must be in the second one. Max 20 coordinates are converted for one conversion. The output is saved as an XLS file to your storage card of the mobile device. You can send this file
